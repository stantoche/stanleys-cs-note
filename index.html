Data representation

1.1 Number systems

1.1.1 How and why computers use binary to represent all forms of data

The computer system is made up of millions of switches. Each of these switches must be in either of two discrete states: ON or OFF. Hence, the binary system is chosen because it only consists of 1s and 0s - where 1 and 0 correspond to ON and OFF.

Note: Switches used in a computer make use of logic gates and are used to store and process data.

At this juncture, it is sufficient to say that computers use binary to represent data because they (computers) are made up of switches, which can be either ON or OFF, and these states correspond to the binary digits 1 and 0.

1.1.2 The denary, binary and hexadecimal number systems

The denary number system

Denary is a base 10 number system. It uses ten separate digits, 0-9, to represent all values. It counts in multiples of 10 - where the right-most digit of any given whole number is in the place-value of 100 or unit, and the place-values increase in powers of 10 from right to left for numbers having more than one digit.

For instance, a three-digit number, 125, will be called out with an attention to the place-values of each of the digits:

10n 102 101 100
Place-values 100 (hundreds) 10 (tens) 1 (units)
Digits 1 2 5

Hence, 125, will be called out as: one hundred and twenty-five.

The binary number system

The binary number system is a base 2 number system. It uses two discrete digits, 1 and 0, to represent all values in the computer system. The digits of any given binary number increase by powers of 2 from right to left. A digit of a binary is referred to as a binary digit (or simply bits). The least significant bit is in the place-value of 20 or 1.

For example, we can represent the denary number, 5, as binary, thus:

2n 23 22 21 20
Place-values 8 (eights) 4 (fours) 2 (twos) 1 (units)
Binary digits 0 1 0 1

Hence, the binary number, 0101 can be called out as: 0 eight, 1 four, 0 two, and 1 unit.

Note: any given form of data must be converted to binary so that they can be processed by the computer.

Bits are stored in special memory slots called registers. Each register is a set of contiguous memory blocks that can only take one bit at a time. The size of a given register is determined by the number of memory blocks therein. Hence, you may come across such conventions as 4-bit registers, 8-bit registers, etc.

We can ascertain the largest possible value that can be stored in a given register by using the formula:

largest possible value in a given register = 2b-1, where b = number of bits in the register.

For example, the largest possible value that can be stored in an 8-bit register can be calculated as follows:

28 - 1 = 256 - 1 = 255

The hexadecimal number system

Hexadecimal is a base 16 number system and uses 16 distinct (nonrepeating) symbols to represent all values. These symbols are the numbers 0-9 and the letters A-F.

For any given hexadecimal number, the digits increase by powers of 16 from right to left, where the least significant hex digit is in the placevalue of 160 or 1 (unit).

For example, we can represent the denary number, 3898 as follows:

16n 162 161 160
Place-values 256 16 1
Hexadecimal digits F (15) 3 (3) A (10)

Hence, we can call out F3A as: 15 two hundred and fifty-sixes, 3 sixteens, and 10 ones (or units).

Note: the reason for using the symbols A-F for the denary numbers 10-15 is so only one symbol is needed for the denary number in hexadecimal.

1.1.3 Converting numbers between denary, binary and hexadecimal

Converting from binary to denary

We can convert a binary number to denary number using any of the methods: summation of binary place-values and sum of powers of 2.

Method 1: Summation of binary place-values

In this method, we place the given binary number, digit-by-digit, in a binary table, and then add the place-values corresponding to the binary digit, 1.

Example:
Convert the binary number 11101110 to denary
Solution:

2n 27 26 25 24 23 22 21 20
D 128 64 32 16 8 4 2 1
B 1 1 1 0 1 1 1 0

Therefore, 11101110 = 128 + 64 + 32 + 8 + 4 + 2 = 238
